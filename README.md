# SmartPark – Real-Time Parking Management System
A modern **web-based smart parking management system** that allows users to monitor parking space availability in real time, reserve parking slots, and manage bookings through an interactive dashboard.

This project demonstrates a **responsive parking management platform** with real-time slot visualization, booking workflow, pricing plans, and payment simulation.

## Features
* Real-time parking slot availability monitoring
* Interactive parking map with multiple floors
* Parking slot selection and booking system
* Vehicle information input during booking
* Dynamic pricing plans (Hourly, Daily, Monthly)
* Payment workflow simulation
* User dashboard for booking management
* Responsive design for mobile and desktop
* Dark mode and theme system
* Parking statistics and analytics display
